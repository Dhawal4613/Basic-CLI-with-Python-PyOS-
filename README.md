# 🖥️ PyOS - Custom Python CLI

PyOS is a powerful custom **Command Line Interface (CLI)** built using Python.  
It allows users to interact with their system, manage files, and execute commands — similar to a mini operating system shell.

---

## 🚀 Features

- 📁 File management (create, delete, copy, move, rename)
- 📂 Directory navigation (`cd`, `pwd`, `ls`)
- 🖥️ System commands (shutdown, restart, lock)
- 🌐 Network tools (IP info, WLAN passwords)
- 📝 File editing (`write`, `append`)
- ⚡ Run system commands (`run`)
- 🎨 ASCII UI using pyfiglet
- 🧠 Custom command parser

---

## 🛠️ Installation

### 1. Install Python
Make sure Python is installed (Only for main.py):

